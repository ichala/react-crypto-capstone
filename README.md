- Reactjs
- Redux
- Css
- Npm


## Live Demo 

you can check the live preview in [Here](https://metacrypt.herokuapp.com/)


## Install

- first, clone the project
  Run

```
  git clone https://github.com/ichala/react-crypto-capstone.git
```

- Install the necessary dependancies
  Run

```
  npm install
```

```
  npm start
```
 ## Test
 
```
  npm test 
```
